# Danphe Experience Hub

A Vite + React multi-app experience demonstrating logistics productivity tracking, finance tools and mindfulness utilities. Each mini-app lives in `src/apps/<domain>` to keep things modular.

## Getting Started

```bash
npm install
npm run dev
```

Visit the printed localhost URL (default `http://localhost:5173`).

## Apps Included

- Productivity Tracker — train driver shift intelligence with CSV export.
- ATO Tax Calculator — FY24/25 aware stepper.
- Home Loan Experience — Westpac-inspired UI with wealth projection.
- Starbucks Ordering — live queue with voice ordering.
- Meditation Counter — mantra recognition via Web Speech API.

Additional folders (Furniture Designer, LinkedIn Mini, etc.) are scaffolded for future work.
